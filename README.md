# secfathy.github.io

Personal blog of Mohammed Fathy — Security Engineer.
Built with [Jekyll](https://jekyllrb.com/) + the `minima` theme, served by GitHub Pages at
**https://secfathy.github.io**.

## New post

Create `_posts/YYYY-MM-DD-title.md`:

```markdown
---
layout: post
title: "Title"
date: 2026-07-15 12:00:00 +0200
tags: [tag1, tag2]
---

body
```

Push to `main` — Pages rebuilds automatically.

## Local preview

```bash
bundle install
bundle exec jekyll serve   # http://localhost:4000
```
